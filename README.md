# Gui Fotos

**Gui Fotos** is an **open-source alternative to Google Photos**, simplicity, privacy, and self-hosting in mind. Perfect for family usage.

---

## Technologies

- **Self-hosted photo storage and sharing**
- **Docker-based deployment**
- **Modern .NET 8+ backend**

## Requirements

Make sure you have the following installed:

- [Docker](https://www.docker.com/)
- [.NET 8 or higher](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
